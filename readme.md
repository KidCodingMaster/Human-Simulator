# Human Simulator v0.0.5b

*This is a VERY simple Human Simulator. This is still in beta version*

****

**How to start**
****

Use this command in the terminal to open virtual environment
`source venv/bin/activate`

Install requirements using this command
`pip3 install -r requirements`


**How to play**
****

You can go to the shop and buy things. You can also work. (you can't actually get a job so you can't work)
